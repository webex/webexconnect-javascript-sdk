# Webex Connect JavaScript SDK

Webex Connect JavaScript SDK provides a messaging framework that enables app developers to embed bidirectional Live Chat Messaging and Web Push Notifications capabilities within their web applications.

## v1.8.x Minified SDK
- Details of all features can be found [here](https://developers.webexconnect.io/docs/overview-of-sdks)
- Follow [JavaScript SDK Quick Start Guide](https://developers.webexconnect.io/docs/javascript-sdk-quickstart-guide) 

## Upgrading from SDK versions v1.7.x and below.
- [Migrate from previous SDKs](https://developers.webexconnect.io/v6.15.0/docs/migrate-from-previous-sdks)
